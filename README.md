# Product Landing Page – MLM Nutrition

This project is a **responsive product landing page** built as part of the
[freeCodeCamp Responsive Web Design Certification](https://www.freecodecamp.org/).

It demonstrates the ability to design and structure a **modern marketing landing page** using semantic HTML, responsive CSS, and UI layout techniques.

---

## 🚀 Live Demo

👉 https://tefol-hub.github.io/product-landing-page/

---

## 💼 Project Highlights

* Built a **fully responsive marketing landing page**
* Designed a structured layout with:

  * Hero section with email signup
  * Product feature breakdown
  * Pricing table section
  * Category-based product grid
* Implemented a **fixed navigation bar with smooth scrolling**
* Integrated **embedded YouTube video content**
* Used **icon-based visual enhancements (Font Awesome)**
* Applied **CSS Grid and Flexbox for responsive layouts**
* Includes **interactive hover animations for UI elements**

---

## 🛠️ Tech Stack

* HTML5 (semantic structure, forms, media embedding)
* CSS3 (Grid, Flexbox, animations, media queries)
* Font Awesome (icons)
* YouTube embed integration
